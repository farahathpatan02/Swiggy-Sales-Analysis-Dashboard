# 🍔 Swiggy Sales Analysis Dashboard

## 📖 Project Overview

This project analyzes Swiggy sales data and presents key business insights through an interactive Excel Dashboard.

The dashboard helps track sales performance, customer ratings, order trends, food category performance, and geographical sales distribution using dynamic filters and visualizations.

## 📈 Dashboard Features

- Monthly Sales Trend Analysis
- Daily Sales Trend Analysis
- Weekly Sales Trend Analysis
- Sales by Food Type
- State-wise Sales Visualization
- Top 5 Cities by Sales
- Quarter-wise Sales, Ratings & Orders Analysis
- Interactive Slicers for:
  - Month
  - Category
  - Restaurant Name

## 🔍 Business Insights

- Bengaluru generated the highest sales revenue.
- Vegetarian food categories contributed the majority of sales.
- Customer ratings remained consistently high with an average rating of 4.34.
- Monthly sales showed stable growth patterns throughout the analyzed period.
- Interactive filters allow users to perform detailed analysis across categories and restaurants.
