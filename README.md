# 🍔 Swiggy Sales Analysis Dashboard

An interactive Excel dashboard built to analyze Swiggy sales data and generate actionable business insights through KPI tracking, trend analysis, and data visualization.

---

## 📊 Dashboard Preview

![Dashboard](Dashboard.png)

---

## 📖 Project Overview

This project analyzes Swiggy sales data using Microsoft Excel to identify sales trends, customer behavior, category performance, and regional contributions.

The dashboard transforms raw transactional data into meaningful insights that can help stakeholders make data-driven decisions.

---

## 🎯 Business Objective

The goal of this project is to:

- Monitor overall sales performance
- Analyze customer ratings and satisfaction
- Identify top-performing cities and states
- Compare food category performance
- Track monthly, weekly, and daily sales trends
- Create an interactive reporting dashboard

## 💡 Key Insights

- Bengaluru generated the highest sales revenue.
- Vegetarian food categories contributed the majority of total sales.
- Customer ratings remained consistently high with an average rating of 4.34.
- Sales trends showed stable performance across multiple months.
- A small number of cities contributed significantly to overall revenue.
